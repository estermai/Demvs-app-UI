DEMVS

Deus Mantém Você Segura

DEMVS é um projeto pessoal de UX/UI desenvolvido no Figma.

O aplicativo foi idealizado para auxiliar usuários em situações de vulnerabilidade por meio de alertas de emergência, compartilhamento de localização em tempo real, check-ins de segurança e modo discreto.

Este projeto foi criado com o objetivo de praticar design de interfaces mobile, fluxos de usuário e experiências voltadas para segurança pessoal.
